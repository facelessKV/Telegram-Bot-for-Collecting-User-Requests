======================================
КАК ЗАПУСТИТЬ TELEGRAM-БОТА ДЛЯ СБОРА ЗАЯВОК
======================================

Этот документ содержит пошаговые инструкции по установке и запуску Telegram-бота на компьютере с Windows или Linux.

======================================
1. ПОДГОТОВКА (ОДИНАКОВО ДЛЯ WINDOWS И LINUX)
======================================

1.1. ПОЛУЧЕНИЕ ТОКЕНА БОТА:

1) Откройте Telegram и найдите бота @BotFather
2) Отправьте команду /newbot
3) Введите имя бота (например, "Бот для сбора заявок")
4) Введите имя пользователя бота (должно заканчиваться на "bot", например "my_request_bot")
5) BotFather выдаст вам токен вида "1234567890:ABCDefGhIJKlmnOPQRstUVwxYZ". Сохраните его!

1.2. ПОЛУЧЕНИЕ ID ЧАТА АДМИНИСТРАТОРА (КУДА БУДУТ ПРИХОДИТЬ ЗАЯВКИ):

1) Найдите в Telegram бота @userinfobot
2) Отправьте ему любое сообщение
3) Бот ответит вам вашим ID - это число, например "123456789"
4) Запишите этот ID - это будет ваш ADMIN_CHAT_ID

======================================
2. УСТАНОВКА НА WINDOWS
======================================

2.1. УСТАНОВКА PYTHON:

1) Скачайте Python 3.10.6 с официального сайта:
   https://www.python.org/downloads/release/python-3106/
   (прокрутите вниз и выберите "Windows installer (64-bit)")

2) Запустите скачанный файл. ВАЖНО: поставьте галочку "Add Python to PATH" внизу окна установщика!

3) Нажмите "Install Now" и дождитесь завершения установки

2.2. СКАЧИВАНИЕ И НАСТРОЙКА БОТА:

1) Создайте на рабочем столе папку "TelegramBot"

2) Разместите файл с кодом бота (например, bot.py) в эту папку

3) Откройте блокнот (Notepad): нажмите Win+R, введите "notepad" и нажмите Enter

4) В блокноте напишите:
   BOT_TOKEN=ваш_токен_бота
   ADMIN_CHAT_ID=ваш_id_чата

   Замените "ваш_токен_бота" на токен, полученный от BotFather
   Замените "ваш_id_чата" на ID, полученный от @userinfobot

5) Сохраните файл как ".env" в ту же папку, где лежит bot.py
   ВАЖНО: при сохранении выберите "Тип файла: Все файлы (*.*)" и введите имя файла именно как ".env"

2.3. УСТАНОВКА ЗАВИСИМОСТЕЙ:

1) Нажмите кнопку "Пуск", введите "cmd" и нажмите Enter

2) В открывшейся командной строке введите команды:
   cd %USERPROFILE%\Desktop\TelegramBot
   pip install aiogram==3.2.0 python-dotenv

3) Дождитесь завершения установки

2.4. ЗАПУСК БОТА:

1) В той же командной строке введите:
   python bot.py

2) Если всё сделано правильно, вы увидите сообщение о запуске бота
   Бот будет работать, пока окно командной строки открыто

3) Чтобы остановить бота, нажмите Ctrl+C в окне командной строки

======================================
3. УСТАНОВКА НА LINUX
======================================

3.1. УСТАНОВКА PYTHON:

1) Откройте терминал (обычно комбинация клавиш Ctrl+Alt+T)

2) Введите следующие команды по очереди:
   sudo apt update
   sudo apt install python3.10 python3-pip python3.10-venv

3.2. СКАЧИВАНИЕ И НАСТРОЙКА БОТА:

1) Создайте папку для бота командой:
   mkdir ~/telegram_bot
   cd ~/telegram_bot

2) Создайте файл с кодом бота:
   nano bot.py

3) Вставьте код бота в открывшийся редактор

4) Нажмите Ctrl+O, затем Enter для сохранения, затем Ctrl+X для выхода

5) Создайте файл окружения:
   nano .env

6) Впишите в файл:
   BOT_TOKEN=ваш_токен_бота
   ADMIN_CHAT_ID=ваш_id_чата

   Замените "ваш_токен_бота" на токен, полученный от BotFather
   Замените "ваш_id_чата" на ID, полученный от @userinfobot

7) Нажмите Ctrl+O, затем Enter для сохранения, затем Ctrl+X для выхода

3.3. СОЗДАНИЕ ВИРТУАЛЬНОГО ОКРУЖЕНИЯ И УСТАНОВКА ЗАВИСИМОСТЕЙ:

1) Создайте виртуальное окружение:
   python3 -m venv venv

2) Активируйте виртуальное окружение:
   source venv/bin/activate

3) Установите зависимости:
   pip install aiogram==3.2.0 python-dotenv

3.4. ЗАПУСК БОТА:

1) Если виртуальное окружение активно (в начале строки будет "(venv)"), введите:
   python3 bot.py

2) Если вы закрыли терминал, то для запуска бота нужно:
   cd ~/telegram_bot
   source venv/bin/activate
   python3 bot.py

3) Чтобы остановить бота, нажмите Ctrl+C в терминале

======================================
4. ИСПОЛЬЗОВАНИЕ БОТА
======================================

1) Найдите своего бота в Telegram по имени пользователя, которое вы задали при создании

2) Отправьте команду /start, чтобы начать заполнение формы

3) Следуйте инструкциям бота для заполнения формы

4) Если нужно отменить заполнение, отправьте команду /cancel

5) После подтверждения заявки, данные будут отправлены на аккаунт администратора (вас)

======================================
5. ВОЗМОЖНЫЕ ПРОБЛЕМЫ И ИХ РЕШЕНИЕ
======================================

ПРОБЛЕМА: "Такого файла или каталога не существует"
РЕШЕНИЕ: Убедитесь, что вы находитесь в правильной папке. Используйте команду cd для перехода в нужную папку.

ПРОБЛЕМА: "ModuleNotFoundError: No module named 'aiogram'"
РЕШЕНИЕ: Запустите установку зависимостей заново: pip install aiogram==3.2.0 python-dotenv

ПРОБЛЕМА: "Не удается сохранить файл .env"
РЕШЕНИЕ: В Windows убедитесь, что при сохранении выбрали "Тип файла: Все файлы (*.*)"

ПРОБЛЕМА: "Бот не отвечает на сообщения"
РЕШЕНИЕ: Убедитесь, что в файле .env правильно указан токен бота

ПРОБЛЕМА: "Админу не приходят заявки"
РЕШЕНИЕ: Проверьте правильность ADMIN_CHAT_ID в файле .env
